# Installation

Clone repo:

```
git clone git@github.com:istinspring/tamrec-ui.git
```

You'll need **nodejs** to install bower & gulp:

```
npm install -g bower
npm install -g gulp
```

next cd into project folder install project dependencies

```
cd tamrec-ui
npm install
bower install
```

run project

```
gulp serve
```
